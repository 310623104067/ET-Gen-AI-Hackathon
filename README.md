# 💰 AI Money Mentor

AI Money Mentor is a web-based application that analyzes a user's financial data and provides insights such as financial score, risk level, FIRE (Financial Independence, Retire Early) target, and personalized financial advice.

This project helps users understand their financial health and make better financial decisions using simple algorithms and an interactive interface.

---

## 🚀 Features

- 📊 Financial Score Calculation based on income, expenses, savings, and EMI  
- ⚠ Risk Level Prediction (Low / Medium / High)  
- 🔥 FIRE Target Calculation for long-term planning  
- 💡 Personalized Financial Advice  
- 🎯 Dynamic output based on user input  
- 🌙 Simple and responsive user interface  

---

## 🧠 Algorithms Used

### 1. Financial Scoring Algorithm
- Calculates score using:
  - Savings Ratio  
  - Expense Ratio  
  - EMI Ratio  

### 2. Risk Prediction
- Rule-based classification:
  - High expense ratio → High risk  
  - Moderate expense ratio → Medium risk  
  - Low expense ratio → Low risk  

### 3. FIRE Calculation
FIRE Target = Annual Expenses × 25

### 4. Growth Simulation (Optional Backend)
- Uses compound growth model to estimate savings growth over time
